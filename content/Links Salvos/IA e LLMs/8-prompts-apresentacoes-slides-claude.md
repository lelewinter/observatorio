# 8 Prompts para Criar Apresentações que Funcionam — Além do Genérico "Faça uma Apresentação para Mim"

## Resumo

Um prompt fraco gera slides fracos. João Martins compartilhou 8 prompts específicos que transformam apresentações genéricas em narrativas estruturadas, com objetivos claros, arco de história, design visual orientado e fechamento impactante — como um roteiro que força o público a lembrar da ideia central mesmo após sair da sala.

## Explicação

**O Problema:**
Dizer "Faça uma apresentação para mim" gera slides genéricos, sem estrutura narrativa ou impacto visual.

**A Solução: 8 Prompts Estruturados**

**1. Defina o Objetivo (Decisão ou Ação)**
```
Prompt: "O objetivo desta apresentação é provocar [uma decisão / uma ação / uma mudança de perspectiva].
Ajude-me a formular uma única ideia-chave que o público ainda se lembrará claramente uma semana
depois de sair da sala."
```

**2. Construa o Arco da História**
```
Prompt: "Crie uma estrutura narrativa para uma apresentação sobre [tema]. A progressão deve ir do problema,
passar pelo obstáculo, chegar ao insight, depois à solução e finalmente ao resultado. Limite a apresentação
a [X] seções principais, sem perder a narrativa."
```

**3. Estrutura Slide por Slide**
```
Prompt: "Crie um roteiro completo de slides para esta apresentação. Em cada slide inclua um título principal,
a ideia central é um elemento de apoio. Elimine tudo o que não seja realmente essencial."
```

**4. Recomendações Visuais**
```
Prompt: "Para cada slide deste roteiro, sugira o formato visual mais adequado: gráfico, diagrama,
imagem ou texto simples. Inclua uma explicação curta de uma linha justificando por que esse formato
funciona melhor para aquele slide específico."
```

**5. Notas para o Apresentador**
```
Prompt: "Escreva notas de apresentação para cada slide. Use um tom natural e fácil de falar em voz alta.
Evite simplesmente repetir o que já está escrito no slide."
```

**6. Revisão Final**
```
Prompt: "Revise este roteiro completo da apresentação: [colar]. Remove tudo o que torne a apresentação
mais lenta, torne a linguagem mais direta e deixe a mensagem geral mais clara, forte e persuasiva."
```

**7. Defini o Público**
```
Prompt: "Esta apresentação será feita para [tipo de público: executivos, estudantes, investidores, etc.].
Identifique as principais prioridades, preocupações e o nível de conhecimento deles sobre [tema],
e sugira como adaptar a mensagem para que realmente"
```

**8. Criar uma Abertura Forte**
```
Prompt: "Ajude-me a criar uma abertura impactante para uma apresentação sobre [tema]. Os primeiros 30
segundos devem prender a atenção. Use estatísticas, histórias ou perguntas provocativas."
```

## Exemplos

**Exemplo completo de fluxo:**

Comece com o Prompt 1 para definir: "O objetivo é convencer investidores a financiar nossa startup de IA"

Depois use Prompt 2 para criar: Problema (mercado fragmentado) → Obstáculo (falta de tooling) → Insight (IA pode unificar) → Solução (nosso produto) → Resultado ($XXM de mercado capturável)

Depois Prompt 3 estrutura em 7 slides com títulos e ideias principais

Prompt 4 sugere: Slide 1 (título+imagem impactante) → Slide 2 (gráfico de fragmentação) → Slide 3 (diagrama da solução) → Slide 4 (caso de uso com foto) → Slide 5 (números em visualização clara) → Slide 6 (roadmap) → Slide 7 (call-to-action)

Prompt 5 adiciona notas naturais para falar em voz alta

Prompt 6 revisão final da coerência

Prompt 7 adapta linguagem para "VCs que buscam espaço TAM grande, com equipe strong e product-market fit"

Prompt 8 cria abertura: "Em 2025, 92% das empresas ainda usam 7+ ferramentas desconectadas para IA. Nós unificamos tudo em uma."
