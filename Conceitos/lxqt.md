# SSH Server

## O que e
Um servidor SSH (Secure Shell) é um programa que permite acesso remoto a um dispositivo através de uma conexão criptografada. Ele fornece uma maneira segura de acessar e controlar um dispositivo remotamente, como um servidor ou um computador.

## Como funciona
O servidor SSH escuta conexões de entrada em uma porta específica (geralmente a porta 22). Quando um cliente SSH se conecta ao servidor, ele estabelece uma conexão criptografada que protege os dados transmitidos de serem interceptados por terceiros. A autenticação é geralmente feita por senha ou chave pública.

## Pra que serve
Os servidores SSH são usados para administração remota de sistemas, transferência segura de arquivos e tunelamento de conexões. Eles são essenciais para administradores de sistemas que precisam gerenciar servidores e outros dispositivos remotamente.

## Exemplo pratico
Um administrador de sistemas pode usar um servidor SSH para acessar um servidor remoto e realizar tarefas de manutenção, como instalar software ou configurar serviços.
