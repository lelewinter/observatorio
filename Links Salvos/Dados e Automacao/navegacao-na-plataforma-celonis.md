# Navegação na Plataforma Celonis

## Resumo
A Celonis Academy oferece treinamento guiado para navegação na plataforma Celonis, focada em Process Mining e otimização de processos empresariais. A URL referenciada aponta para um vídeo introdutório de orientação da interface da plataforma.

## Explicação
A plataforma Celonis é uma solução de Process Mining que permite às organizações capturar, analisar e melhorar processos de negócio a partir de dados de sistemas como SAP, Salesforce e outros ERPs. Navegar pela plataforma é o primeiro passo para extrair valor analítico dos dados de processo.

A Celonis Academy é o braço educacional oficial da Celonis, disponibilizando trilhas de aprendizado que cobrem desde a navegação básica da interface até a criação de análises avançadas com PQL (Process Query Language). O vídeo referenciado nesta nota trata especificamente de como se orientar dentro do ambiente da plataforma — incluindo áreas como Studio, Apps, e os diferentes módulos de análise.

Do ponto de vista prático, entender a navegação da plataforma é pré-requisito para qualquer uso produtivo do Celonis: sem dominar onde estão as ferramentas de análise, os conectores de dados e os dashboards, o usuário não consegue construir ou interpretar os chamados Process Views (visões de processo). A interface é organizada em torno de workspaces, packages e análises, o que exige familiaridade inicial.

Como não há notas relacionadas no vault, esta nota serve como ponto de entrada para um cluster temático de Process Mining e ferramentas de análise de processos que pode ser expandido futuramente.

## Exemplos
1. **Onboarding corporativo**: Um analista de processos recém-contratado utiliza a Celonis Academy para aprender a navegar até os dashboards de conformidade de processos P2P (Procure-to-Pay).
2. **Certificação profissional**: Um consultor usa o vídeo de navegação como base antes de avançar para módulos de PQL e construção de análises customizadas.
3. **Exploração de Apps**: Um gestor de operações aprende a localizar e abrir Celonis Apps pré-construídos para monitorar KPIs de Order-to-Cash sem necessidade de configuração manual.
