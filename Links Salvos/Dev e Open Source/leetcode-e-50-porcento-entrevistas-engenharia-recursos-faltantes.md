# Preparar para 50% de Entrevistas com Leetcode e Algoritmos

## O que é
Leetcode cobre ~50% de perguntas técnicas em entrevistas. Falta: system design, comportamento, negociação.

## Como implementar
**Plano de estudo:**
- Weeks 1-2: Arrays/Strings (25 easy problems)
- Weeks 3-4: Trees/Graphs (20 medium problems)
- Weeks 5-6: Dynamic Programming (15 hard problems)
- Weeks 7-8: Mock interviews (LeetCode + Pramp)

**Stack:** Python + Leetcode Premium

## Armadilhas
1. Memorizar ≠ Entender. Focus em padrões
2. Praticar todas as variações de um problema
