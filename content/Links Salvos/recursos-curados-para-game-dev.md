# Recursos Curados para Game Dev

## Resumo
Listas curadas de recursos para desenvolvimento de jogos reúnem ferramentas, engines, tutoriais, assets e referências em um único repositório público, servindo como ponto de entrada estruturado para a área.

## Explicação
Desenvolvimento de jogos é uma disciplina ampla que envolve programação, design, arte, áudio, física, IA e publicação. A fragmentação de recursos na internet torna difícil para iniciantes e intermediários encontrar materiais confiáveis e relevantes. Repositórios curados, como o GameDev-Resources no GitHub (github.com/Kavex/GameDev-Resources), resolvem esse problema ao agregar e categorizar fontes validadas pela comunidade.

Esse tipo de lista geralmente organiza recursos por categoria: engines (Unity, Godot, Unreal), linguagens de scripting, ferramentas de arte 2D/3D, geradores de som, plataformas de publicação, comunidades e cursos. A curadoria coletiva garante que links desatualizados sejam removidos e que novos recursos relevantes sejam adicionados via pull requests.

Para quem está aprendendo ou estruturando um pipeline de desenvolvimento, esse tipo de repositório funciona como um mapa do ecossistema. Ele não substitui documentação oficial nem cursos aprofundados, mas oferece visibilidade sobre o que existe e quais ferramentas a comunidade considera essenciais.

Do ponto de vista de aprendizado, listas curadas são um artefato de conhecimento coletivo — representam o consenso prático da comunidade sobre o que vale a pena estudar, diferente de tutoriais individuais que refletem apenas uma perspectiva.

## Exemplos
1. **Escolha de engine**: Um desenvolvedor iniciante consulta a lista para comparar Godot vs Unity antes de começar seu primeiro projeto.
2. **Pipeline de arte**: Um programador sem background artístico usa a seção de ferramentas de arte para encontrar geradores de sprites e tilesets gratuitos.
3. **Publicação indie**: Um desenvolvedor solo usa a seção de plataformas para identificar onde lançar seu jogo além da Steam (itch.io, Game Jolt, etc.).
