# Projetos GitHub em Crescimento - Lista dos Mais Rápidos em Março

## Resumo

Compilação dos projetos GitHub crescendo mais rapidamente em março de 2026, focando em IA e agentes. Repositórios que ganham stars diariamente e indicam tendências do ecossistema: OpenClaw (122K stars), Superpowers (30.7K), RuView (30.4K), MirroFish (17K), Airi (10K) e outros. É como "Top 40 de música" mas para código — lista de qual código toda comunidade está ouvindo (usando), aponta futuro do que deve aprender.

## Explicação

OpenClaw — assistente de IA pessoal (122K stars) — roda 24/7 em qualquer sistema operacional permitindo automação contínua e execução de agentes em background. Superpowers — framework agentic (30.7K stars) — fornece habilidades modulares plug-and-play para agentes de IA, padrão emergente para construção de agentes escaláveis.

**Analogia:** GitHub trending é como "GPS de mercado" — mostra para onde toda gente está migrando. 122K stars em OpenClaw não é por acaso — significa que "IA pessoal 24/7" é problema que comunidade decidiu resolver urgentemente. Você quer saber o que vai ser importante? Veja trending de GitHub.

RuView — detecção de pose humana por WiFi (30.4K stars) — transforma sinais WiFi regulares em detecção de pose humana em tempo real sem câmeras ou sensores adicionais. MirroFish — mecanismo de inteligência de swarm (17K stars) — prevé praticamente qualquer coisa usando inteligência coletiva. Airi — companheiro de IA auto-hospedado (10K stars) — companheiro com voz real-time que roda em sua própria máquina com criptografia local.

**Profundidade:** O que trending mostra? Que privacidade é critical — Airi auto-hospedado ganha stars porque cloud services não confiam. Que autonomia importa — OpenClaw 24/7 porque "on demand" virou bottleneck. Que modularidade vence — Superpowers porque monolith é passado. Trending = mercado votando.

Claude-Code-Best-Practice (11.8K stars) — repositório de melhores práticas para construir com Claude Code com padrões recomendados, exemplos, guias. PI-Mono — toolkit completo (11.8K stars) — inclui CLI unificado, interface web, bot para Slack. Deer-Flow — supera gent de pesquisa (10.4K stars) — open-sourced por ByteDance, faz pesquisa autônoma e cria conteúdo próprio. Learn-Claude-Code (9K stars) — curso de como construir clone de Claude Code do zero.

Padrões observáveis: agentes autônomos (múltiplos projetos focam em agentes que rodam 24/7), privacidade (tendência de auto-hospedagem e controle local), integração modular (frameworks plug-and-play), IA em tempo real (voz, detecção, interação), open-source corporativo (grandes empresas liberando código).

## Exemplos

Crescimento acelerado reflete movimento em direção a: IA mais autônoma e contínua, maior controle e privacidade, ferramentas especializadas para agentes, educação e democratização de conhecimento.

Estes projetos indicam: onde a comunidade está investindo, que problemas estão sendo resolvidos, tecnologias emergentes, padrões futuros de desenvolvimento IA.
