# Estudar Algoritmos com Livro MIT "Algorithms" (700 páginas)

## O que é
Livro definit de algorithms do MIT. Cobre: análise (Big-O), sorting, graphs, DP, NP.

## Como implementar
**Estrutura de estudo:**
- Introdução + Big-O (semana 1)
- Sorting algorithms (merge, quick, heap) - semana 2
- Data structures - semana 3
- Graph algorithms (BFS, DFS, Dijkstra) - semanas 4-5
- Dynamic Programming - semana 6
- Problemas NP-complete - semana 7

**Prática:** resolver 5 problemas por capítulo

## Stack e requisitos
- Livro físico/PDF (200 reais/USD)
- Python/C++ para implementar
