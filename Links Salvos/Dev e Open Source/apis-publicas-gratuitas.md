# APIs Públicas Gratuitas

## Resumo
Existem repositórios curados com centenas de milhares de APIs públicas e gratuitas, cobrindo domínios como clima, finanças, IA, mapas e entretenimento, eliminando custos de aquisição de dados no desenvolvimento de aplicações.

## Explicação
APIs públicas gratuitas são interfaces de programação disponibilizadas sem custo por governos, empresas ou comunidades open source, permitindo que desenvolvedores consumam dados e funcionalidades prontas sem infraestrutura própria. A existência de repositórios curados com mais de 320.000 entradas verificadas representa uma mudança significativa no custo de entrada para desenvolvimento de software — praticamente qualquer domínio de dados pode ser acessado sem investimento inicial.

A categorização por domínio (clima, finanças, notícias, esportes, cripto, IA/ML, geolocalização, entretenimento) permite que desenvolvedores prototipen produtos completos usando apenas integrações externas, postergando ou eliminando a necessidade de coleta e armazenamento próprio de dados. Isso é especialmente relevante em contextos de MVP (Produto Mínimo Viável), onde velocidade e custo baixo são prioritários.

Do ponto de vista técnico, o uso de APIs públicas implica dependência de terceiros, limites de requisição (rate limits), possível instabilidade e ausência de SLA. Portanto, embora sejam ideais para prototipagem e projetos independentes, aplicações críticas em produção devem avaliar a confiabilidade e longevidade de cada endpoint antes de adotá-lo como dependência central.

## Exemplos
1. **Prototipagem de app financeiro**: integrar APIs gratuitas de cotações de criptomoedas e câmbio para construir um dashboard sem banco de dados próprio.
2. **Aplicação de IA**: consumir APIs públicas de machine learning (ex: inferência de modelos) para adicionar funcionalidades inteligentes sem treinar modelos próprios.
3. **App de clima ou geolocalização**: usar APIs governamentais de dados meteorológicos e mapas abertos (ex: OpenWeatherMap, OpenStreetMap) para serviços baseados em localização.
